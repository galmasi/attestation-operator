Keylime CRDs installed.
API group is attestation.keylime.dev.

Run "kubectl get crd | grep keylime" to see the list of available CRDs.

NOTE: Updating this Helm release will update the CRDs. It can lead to data loss if some of the field names are changed or fields are removed!

NOTE: Deleting this Helm release will delete CRDs. It will lead to deleting all instances of the CRs created.
